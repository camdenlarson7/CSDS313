# CSDS 313 Coursework Repository

## Overview
This repository contains code, data, and analysis for assignments in CSDS 313. It covers data cleaning, visualization, and statistical modeling for multiple real-world datasets, including music albums, anime ratings, algorithm performance, running times, city populations, call details, etc. The code demonstrates best practices in data science, including reproducible analysis, clear visualizations, and model fitting.
